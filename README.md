About This project is based on investment training ideas from Li Xiaolai's book, *The Road to Wealth Freedom*. Thanks to Mr. Li Xiaolai for the idea.
This project allows you to practice investing. You input the assets you want to invest in, and the program automatically writes the closing price of the 1st on the 2nd of each month. After you press the send button, it sends your investment information to your email address. It uses Python, with Streamlit as the front-end.
Currently, I've only set it to update once a month. I hope it helps you.
